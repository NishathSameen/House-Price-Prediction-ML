# 🏠 House Price Prediction Using Linear Regression

Successfully completed **Task 01** during my **Machine Learning Internship at SkillCraft Technology**.

## 📌 Project Overview
This project predicts house prices using Linear Regression and the Kaggle House Prices dataset.

## 📄 Files

### houseprediction.py
Core machine learning model implementation using Linear Regression. It performs:
- Data loading and preprocessing
- Model training and evaluation
- House price prediction
- Data visualization

### app.py
Interactive dashboard built using Streamlit and Plotly. It provides:
- Sidebar sliders for user input
- Real-time predictions
- Interactive graphs and visualizations

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Plotly
- Streamlit

## 📊 Dataset
Kaggle Dataset:
https://www.kaggle.com/competitions/house-prices-advanced-regression-techniques/data
